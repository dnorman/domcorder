# Getting Started with DomCorder

DomCorder captures web pages as "videos" that can be played back with full fidelity.

## Prerequisites

- [Bun](https://bun.sh) (latest version)
- Modern web browser

## Quick Start

```bash
cd domcorder
bun install

# Start development (builds injection script + starts server)
bun run dev
```
